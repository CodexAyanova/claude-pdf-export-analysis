Claude PDF Export Analysis

Overview-

This repository documents a failure discovered in Claude's PDF export functionality.

The generated PDF contained:

- Overlapping text
- Clipped paragraphs
- Incorrect page breaks

The project includes:

- Failure analysis
- Product Requirements Document
- Evaluation benchmark
- Supporting evidence
